## 1. DEEP DIVE: The School Management Tech Stack

The B2B EdTech space has three distinct layers, each with different economics:

### Layer 1: School Operations (SIS/ERP)
**What it does:** Admissions, fees, attendance, transport, timetable, parent communication
**Why it matters:** Schools can't function without this â€” it's mission-critical infrastructure
**Key insight:** Most Indian schools still use Excel + WhatsApp. The TAM is massive but sales cycles are brutal (decisions made by trust committees, not individual buyers)

### Layer 2: Classroom Tech (LMS/Content)
**What it does:** Lesson planning, content delivery, assessments, homework
**Why it matters:** This is where learning outcomes actually happen
**Key insight:** Content is NOT the moat â€” distribution is. Whoever owns the teacher relationship wins.

### Layer 3: Outcomes Layer (Assessment/Analytics)
**What it does:** Testing, diagnostics, learning gap identification, board exam prep
**Why it matters:** Parents pay for results, not platforms
**Key insight:** The real opportunity is connecting all three layers â€” no one has done this well in India yet.

**Tech Stack Reality Check:**
- Most schools run 4-5 different disconnected tools
- Data doesn't flow between them
- Teachers spend 2+ hours daily on admin work that could be automated
- The integration opportunity is worth more than any individual layer
